# Camel Club Narvik 2026 - Event Program PWA

A Progressive Web App displaying the event program for Camel Club Narvik 2026 (April 23-26).

## Features

- Progressive Web App (PWA) - can be installed on mobile devices
- Offline support via Service Worker (auto-generated by Vite PWA)
- Responsive design for all devices
- Beautiful background image
- Clean, readable schedule layout
- Built with Vite for fast development and optimized builds

## Development

### Prerequisites

- Node.js (v18 or higher)
- pnpm (install with `npm install -g pnpm`)

### Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
# Create optimized production build
pnpm build

# Preview production build locally
pnpm preview
```

The production files will be in the `dist/` directory.

## Deployment to GitHub Pages

### Option 1: Manual Deployment

1. Build the project:
   ```bash
   pnpm build
   ```

2. Create a new repository on GitHub

3. Push the built files:
   ```bash
   git add dist -f
   git commit -m "Deploy to GitHub Pages"
   git subtree push --prefix dist origin gh-pages
   ```

4. Enable GitHub Pages:
   - Go to repository Settings → Pages
   - Source: "gh-pages" branch
   - Click "Save"

### Option 2: GitHub Actions (Recommended)

Add this workflow file at `.github/workflows/deploy.yml`:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ]

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          cache: 'pnpm'
      
      - name: Install dependencies
        run: pnpm install
      
      - name: Build
        run: pnpm build
      
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./dist
```

Then:
1. Push your code to GitHub
2. Enable GitHub Pages (Settings → Pages → Source: "gh-pages" branch)
3. The app will auto-deploy on every push to main

## Installing as PWA

Once deployed, users can install the app on their devices:

- **iOS/Safari**: Tap the Share button, then "Add to Home Screen"
- **Android/Chrome**: Tap the menu (⋮), then "Add to Home Screen" or "Install App"
- **Desktop/Chrome**: Click the install icon in the address bar

## Tech Stack

- **Vite**: Fast build tool and dev server
- **Vite PWA Plugin**: Automatic service worker and manifest generation
- **Workbox**: Advanced service worker caching strategies
